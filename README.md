
## REACT TASK MANAGER

This App allows to add/update/delete Tasks,and it takes title,description,start-time for every task.When scheduled time is reached for any task,a push notification will appear on browser showing scheduled time for a specific task and its details

All the code is written as per Airbnb javascript conventions

In the project directory, you can run:

### START SCRIPT

$ npm install

$ npm start

Runs the app in the development mode.<br>
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.<br>
You will also see any lint errors in the console.
